# State Management Exam – React

This project was created as an **exam task** to demonstrate working with **5 different state management libraries** in React.

## State Managers Used

1. ✅ React Context API
2. ✅ Redux Toolkit
3. ✅ Jotai
4. ✅ MobX
5. ✅ Zustand

Each state manager is implemented separately to show understanding of:

* State creation
* Updating state
* Async logic (where applicable)
* Component interaction

## Technologies

* React
* Vite
* JavaScript
* Material UI
* REST API

## How to run the project

```bash
npm install
npm run dev
```

## Purpose

This project was built to **demonstrate knowledge of multiple state management approaches in React** for educational and exam purposes.
